# Art direction notes for high-taste image-based PPTs

## Goal

Make the slides feel art-directed, contemporary, and graphically sophisticated.

## Positive cues

Use language such as:

- minimalist modern flat design
- Apple keynote-like restraint
- Bauhaus geometric clarity
- constructivist composition
- Swiss / International Typographic Style
- contemporary editorial design
- Swiss-grid discipline
- asymmetrical but balanced layout
- refined information design
- premium keynote aesthetic
- poster-like typography
- restrained palette with intentional accent color
- tactile but subtle texture
- conceptual visual metaphor
- elegant hierarchy and negative space

## Negative cues to avoid

- stock-photo business cliché
- cheap corporate template
- default SaaS blue gradient everywhere
- noisy infographic collage
- random floating icons
- childish illustration style unless explicitly appropriate
- gratuitous 3D effects, glows, bevels, shadows
- generic “AI brain” symbol with no concept
- skeuomorphic mockups or photorealistic stock scenes when a flat diagram would
  communicate better

## Default international direction

When the user does not name a style, prefer a globally legible modern deck:
flat vector-like composition, crisp hierarchy, restrained color, large
typography, geometric shapes, high whitespace, and presentation-native visuals.
Good references include Apple launch-keynote restraint, Bauhaus geometric
systems, constructivist poster composition, and Swiss typographic discipline.

## Slide-type suggestions

### Cover
A single strong visual metaphor, sparse text, bold hierarchy, memorable silhouette.

### Framework / method
Diagrammatic clarity, graphic rigor, modular structure, deliberate labeling.

### Evidence / result
One dominant visual story, not a dashboard dump. Use annotations only where they truly help.

### Roadmap / timeline
Elegant directional flow, calm spacing, strong sequencing rhythm.

### Closing
Emotionally resolved, visually calm, high confidence, not a bullet list screenshot.
